# OdaabNote 스터디 가이드

## 이 프로젝트가 하는 일

**9급 공무원 컴퓨터일반** 과목의 기출문제를 관리하고, 모의고사를 풀 수 있는 웹 앱입니다.

### 왜 만들었나 (프로젝트 의의)

- **수동 입력의 한계**: 기출문제를 하나씩 타이핑하면 시간이 오래 걸리고 실수도 생김
- **AI 활용**: 이미지 한 장이면 OCR + AI가 정답·해설·키워드를 자동 추출 → **문제 등록 시간 대폭 단축**
- **체계적 관리**: 태그(초빈출, 별표100개 등)로 분류해 **어떤 문제를 놓쳤는지** 한눈에 파악
- **실전 연습**: 랜덤/과목별/전체 모의고사로 **실제 시험처럼** 풀어볼 수 있음

---

## 사용 방법 (요약)

### 1. 문제 등록

| 방법 | 설명 |
|------|------|
| **이미지 업로드** | 문제 사진 업로드 → Vision OCR → Gemini가 정답·해설·키워드 자동 분석 → 저장 |
| **JSON 일괄 등록** | PDF 등에서 추출한 기출문제 JSON을 `POST /api/problems/import`로 한 번에 등록 |

### 2. 태그로 분류·검색

- **고정 태그**: 초빈출, 별표100개, 지엽적, 기출, 통암기
- **유저 태그**: 회원가입 시 본인 이름으로 태그 생성 → 내가 등록한 문제만 모아보기
- 태그별로 문제 목록 조회 가능

### 3. 모의고사

- **랜덤**: 전체에서 랜덤 출제
- **과목별**: 특정 과목만 출제
- **전체**: 전체 과목 통합 모의고사

### 4. 댓글

- 각 문제에 댓글 달아 스터디원끼리 토론 가능

## 로컬에서 실행하기 (개발자용)

```bash
# 1. 환경변수
cp .env.example .env   # GEMINI_API_KEY, GCP 설정 등 입력

# 2. MySQL
docker-compose up -d mysql

# 3. 백엔드
./gradlew bootRun      # http://localhost:8080

# 4. 프론트엔드
cd frontend && npm install && npm run dev   # http://localhost:5173
```
